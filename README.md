## Description

API build with [Nest](https://github.com/nestjs/nest) framework, using [Pactum](https://pactumjs.github.io/) for e2e testing and [Prisma](https://www.prisma.io/) ORM.

## Installation

```bash
$ yarn install
$ cp .env.example .env
$ cp .env.test.example .env.test
```


## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# e2e tests
$ yarn run test:e2e
```
